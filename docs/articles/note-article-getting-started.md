# 【初心者向け】Miyabiで始める自律型AI開発 - 5分でセットアップ完了

## プログラミング初心者でも大丈夫！AIが全部やってくれます

こんにちは、Miyabi開発者の林です。

「AIでコードを書いてほしいけど、どうすればいいの？」

そんな方のために、**5分でできるセットアップガイド**を書きました。

---

## Miyabiとは？

**Issue（課題）を書くだけで、AIがコードを書いてくれるツール**です。

```
あなたがやること：
📝 「ログイン機能を作って」と書く

AIがやること：
🤖 コードを書く
🧪 テストを作る
📤 PRを出す
```

たった10-15分で完成します。

---

## 必要なもの

1. **パソコン** (Windows / Mac / Linux)
2. **Node.js** (v18以上)
3. **GitHubアカウント**

それだけです。

---

## セットアップ手順

### ステップ1: Node.jsのインストール

まだの方は、以下からダウンロード：
https://nodejs.org/

「LTS」をダウンロードしてください。

### ステップ2: Miyabiのインストール

ターミナル（コマンドプロンプト）を開いて：

```bash
npm install -g miyabi
```

### ステップ3: プロジェクト作成

```bash
npx miyabi
```

メニューが表示されます：

```
🌸 Miyabi へようこそ！

? 何をしますか？
❯ 🆕 新規プロジェクト作成
  📦 既存プロジェクトに追加
  📊 ステータス確認
  🩺 診断
```

「新規プロジェクト作成」を選んでください。

### ステップ4: GitHubログイン

自動でブラウザが開きます。GitHubにログインしてください。

### ステップ5: 完了！

```
✅ Repository created
✅ 53 labels configured
✅ 12 workflows deployed
✅ Projects V2 linked

🎉 セットアップ完了！
```

---

## 使い方

### 1. Issueを書く

GitHubで「New Issue」をクリック。

例：
```
タイトル: ログイン機能を追加

内容:
- メールアドレスとパスワードでログイン
- ログイン状態を保持する
- ログアウトボタンを追加
```

### 2. 待つ

AIが自動で作業を開始します。

### 3. PRを確認

10-15分後、PRが届きます。

## よくある質問

### Q: プログラミングの知識は必要？

**A: いいえ。** 日本語でIssueを書くだけです。

### Q: 無料で使える？

**A: はい。** Miyabi自体は無料です。
（GitHub Actionsの無料枠を使用）

### Q: Windowsでも使える？

**A: はい。** v0.19.0からWindowsに完全対応しました。

### Q: エラーが出たら？

**A: `npx miyabi doctor`を実行してください。**
問題を診断してくれます。
