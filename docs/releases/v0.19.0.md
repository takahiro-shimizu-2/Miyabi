# Miyabi v0.19.0 Release Notes

**Release Date:** 2025-12-19

---

## Highlights

### 7 Claude Skills

Miyabi v0.19.0 introduces **7 reusable Claude Skills** - workflow templates that integrate seamlessly with Claude Code.

| Skill | Description |
|-------|-------------|
| `code-reviewer` | Comprehensive code review with 100-point quality scoring |
| `commit-helper` | Conventional Commits message generation |
| `test-generator` | Unit test generation for TypeScript/JavaScript |
| `doc-generator` | JSDoc/README/API documentation generation |
| `refactor-helper` | Safe code refactoring with step-by-step guidance |
| `skill-creator` | Meta-skill for creating custom skills |
| `autonomous-coding-agent` | CLI integration for autonomous workflows |

### Windows Cross-Platform Support

Full Windows compatibility with:
- `cross-platform.ts` utility for shell-agnostic command execution
- Automatic shell detection (cmd.exe on Windows)
- `.gitattributes` for consistent line endings

### Dependencies Updated

Major dependency updates with breaking change handling:

| Package | Before | After |
|---------|--------|-------|
| `@anthropic-ai/sdk` | 0.65.0 | 0.71.2 |
| `@octokit/rest` | 20.x | 21.1.1 |
| `@octokit/graphql` | 7.x | 8.2.1 |
| `vitest` | mixed | 3.2.4 (unified) |
| `@types/node` | 20.x | 22.15.0 |

## Installation

```bash
# New installation
npm install -g miyabi@0.19.0

# Upgrade
npm update -g miyabi
```

## Using Skills

Skills auto-activate based on triggers:

```bash
# In Claude Code
claude "review this code"        # Activates code-reviewer
claude "write tests for this"    # Activates test-generator
claude "commit these changes"    # Activates commit-helper
```

## Breaking Changes

### @octokit/rest 21.x

```typescript
// Before (v20.x)
const octokit = new Octokit({ auth: token });

// After (v21.x) - Same API, improved types
const octokit = new Octokit({ auth: token });
```

### @anthropic-ai/sdk 0.71.x

```typescript
// Before (v0.65.x)
const response = await anthropic.messages.create({...});

// After (v0.71.x) - Same API, new features
const response = await anthropic.messages.create({...});
```

## Migration Guide

### From v0.18.x

1. Update package:
   ```bash
   npm update -g miyabi
   ```

2. Skills are auto-installed - no action needed

3. Windows users: Existing projects work without changes
